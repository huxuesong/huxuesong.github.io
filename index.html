IOS微信精选开发总结

#导语
       发表是最好的记忆。最近在论坛上看到一个关于微信精选的android开发博文微信精选，感觉这是新闻类app的典型demo,所以决定自己做一个IOS版的，其间遇到了一些问题，并且用到了一些第三方框架，在此和大家分享一下我的学习体会，希望对学习iOS的有些帮助。

#开发

##功能介绍





       主要功能是对后台提供的热门微信数据进行解析（数据是聚合网站提供的，想学大数据的可以绕过了）并在cell上进行显示，点击cell跳转到对应的微信公众号文章（如上图所示）。




       哈哈，是不是感觉功能很简单，大神可以绕过了，对于新手来说，接下来介绍一些干货。

##技术亮点

###一些第三方框架的运用

###对JSON数据进行解析
       一般情况下通过url获得到的数据如下json数据（点我）。json数据转化规则，{}转化为NSDictionary,[]转化为NSArray。具体代码如下。

+(NSMutableArray *)WCsWithData:(NSData *)data{
    NSDictionary *WCs= [NSJSONSerialization JSONObjectWithData:data            options:NSJSONReadingMutableLeaves error:nil];
 NSDictionary *result = WCs[@"result"];
NSArray *lists = (NSArray *)result[@"list"];
NSMutableArray *arrayM = [NSMutableArray array];
for (NSDictionary *dict in  lists) {
    [arrayM addObject:[self WCWithDict:dict]];
}
return arrayM;
}

#总结
       这个简单项目大概花了一天时间（其间包括搓炉石和斗地主的时间），遇到的坑主要是autolayout的布局问题，由于接口是别人提供的，一些功能比如搜索之类的也没办法添加了（认识到了后端的重要性）.好了本项目的地址是 https://github.com/z1017371914/cellDemo。如果你对我表示支持的话，请star或fork一下，给我继续做下去的动力。

在MAC上配置OpenCV环境

##缘起
最近有个工程要用到python里的OpenCV库，但是由于MAC最新系统（OSX 10.11）安全性太高，直接用pip install 安装会出错，安装的过程遇到很多坑，在这里总结一下，与大家分享。

##过程

  1. 安装brew

  2. 安装python（直接在官网下载安装包）点击.dmg文件，安装python。

在应用程序列表中会添加IDEL、pythonLauncher、buildapplet三个应用。

同时，系统默认的python被替换为新版本，新版本安装路径。

/Library/Frameworks/Python.framework/Versions/2.7/
/Library/Frameworks/Python.framework/Versions/2.7/lib/python2.7/site-package    #官方 python安装包路径



/System/Library/Frameworks/Python.framework/Versions/2.7
/Library/Python/2.7/site-packages     #osx python安装包路径

3.安装OpenCV

如果直接通过brew install opencv安装的话，会出现什么linpng16.16 动态库版本太低的问题，那么我们就得安装低版本的opencv,这里我安装的是opencv 2.4.10，通过源代码安装。

     这里通过下载下来代码安装
  cd /usr/local/opencv-2.4.10
mkdir release
cd release

cmake -D PYTHON_EXECUTABLE=/Library/Frameworks/Python.framework/Versions/2.7/    bin/python \
     -D PYTHON_PACKAGES_PATH=/Library/Frameworks/Python.framework/Versions/2.7/    lib/python2.7/site-packages \
 -D INSTALL_PYTHON_EXAMPLES=ON \
 -D PYTHON_INCLUDE_DIR=/Library/Frameworks/Python.framework/Versions/2.7/    Headers \
 -D PYTHON_LIBRARY=/Library/Frameworks/Python.framework/Versions/2.7/lib/    libpython2.7.dylib \
..

make -j8
make install

4.这时你又会发现报编译错误，这里下载的opencv源代码有错误，需要改两处地方（你说坑爹不坑爹）。

在modules/legacy/src/calibfilter.cpp 中做如下更改



5.这回make -j8顺利编译通过，然后再make install 就能正常使用opencv了。

#结语
这是我找到的在最新的OSX系统上安装opencv的方法，如果你有什么更好的方法，请与大家分享交流。

在CentOS7上部署MQTT服务器 

Step 1: Add the CentOS 7 mosquitto repository
$ cd /etc/yum.repos.d
$ wget http://download.opensuse.org/repositories/home:/oojah:/mqtt/    CentOS_CentOS-7/home:oojah:mqtt.repo
$ sudo yum update
Step 2: Install mosquitto & mosquitto-clients

$ sudo yum install mosquitto
$ sudo yum install mosquitto-clients
Step 3: Run mosquitto

$ sudo su
# /usr/sbin/mosquitto -d -c /etc/mosquitto/mosquitto.conf > /var/log/    mosquitto.log 2>&1
Step 4: Test mosquitto_sub & mosquitto_pub

$ mosquitto_sub -h [server_address] -t /[topic]
$ mosquitto_pub -h [server_address] -t /[topic] -m “[message]”
