IOS微信精选开发总结

#导语
       发表是最好的记忆。最近在论坛上看到一个关于微信精选的android开发博文微信精选，感觉这是新闻类app的典型demo,所以决定自己做一个IOS版的，其间遇到了一些问题，并且用到了一些第三方框架，在此和大家分享一下我的学习体会，希望对学习iOS的有些帮助。

#开发

##功能介绍





       主要功能是对后台提供的热门微信数据进行解析（数据是聚合网站提供的，想学大数据的可以绕过了）并在cell上进行显示，点击cell跳转到对应的微信公众号文章（如上图所示）。




       哈哈，是不是感觉功能很简单，大神可以绕过了，对于新手来说，接下来介绍一些干货。

##技术亮点

###一些第三方框架的运用

platform :ios,’7.1’

###对JSON数据进行解析
       一般情况下通过url获得到的数据如下json数据（点我）。json数据转化规则，{}转化为NSDictionary,[]转化为NSArray。具体代码如下。

+(NSMutableArray *)WCsWithData:(NSData *)data{
    NSDictionary *WCs= [NSJSONSerialization JSONObjectWithData:data            options:NSJSONReadingMutableLeaves error:nil];
 NSDictionary *result = WCs[@"result"];
NSArray *lists = (NSArray *)result[@"list"];
NSMutableArray *arrayM = [NSMutableArray array];
for (NSDictionary *dict in  lists) {
    [arrayM addObject:[self WCWithDict:dict]];
}
return arrayM;
}

#总结
       这个简单项目大概花了一天时间（其间包括搓炉石和斗地主的时间），遇到的坑主要是autolayout的布局问题，由于接口是别人提供的，一些功能比如搜索之类的也没办法添加了（认识到了后端的重要性）.好了本项目的地址是 https://github.com/z1017371914/cellDemo。如果你对我表示支持的话，请star或fork一下，给我继续做下去的动力。
